🍔 Hamburgueria Web

Sistema web desenvolvido para gerenciamento de uma hamburgueria, permitindo o controle de pedidos, lanches, bebidas e estoque de forma eficiente.

O projeto foi criado com foco em organização de vendas, automação de processos, melhor controle de Estoque, maior Lucro sobre a empresa, Relatórios de Desempenho, Sistema de Análise,  e facilidade de uso para administradores do sistema.

📌 Funcionalidades

✅ Cadastro de lanches
✅ Cadastro de bebidas
✅ Controle de estoque
✅ Registro de vendas
✅ Sistema de login e autenticação
✅ Controle de acesso por usuário (admin)
✅ Listagem e edição de produtos
✅ Dashboard com informações de vendas e Desempenho dos Garçoms
